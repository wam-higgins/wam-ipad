# wam-ipad
iPad interface for the Higgins Armory collection at the Worcester Art Museum

This repository is used for all kiosks, change the starting page to change the kiosk contents.

## Loading in Kiosk Pro

- Copy all videos to a directory called "videos" to the root of Kiosk Pro
- Copy this project's directory "wam-ipad" to the root of Kiosk Pro

## Set the start page

- The start page determines which iPad experice is used.

armor.html -- The Armor Kiosk
sword.html -- The Sword Kiosk
index.html -- The Explore Kiosk

You could also use bts.html or featured.html as well.
